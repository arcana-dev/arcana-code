## Getting Started

1. Clone the repository:

# Install frontend dependencies
```console
cd agent-arcana
npm install
```

2. Set up environment variables:

# Frontend (.env.local)
GEMINI_API_KEY="your api key"


# Frontend (in a new terminal)
```console
cd agent-arcana
npm run dev
```
